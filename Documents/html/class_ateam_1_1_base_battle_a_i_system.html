Common.MOVE_TYPE  type 
)

指定したアクターの移動

指定したアクターIDのオブジェクトを移動させる
移動はステージのブロックにそってのみ行われる
斜め移動はなし、上下左右のみ移動命令が行える
１ブロックの移動スピードはアクターに依存する
移動中のアクター、もしくは制御下にないアクターは移動命令を受け付けない

戻り値
移動可否
引数
actorId移動させたいアクターID
